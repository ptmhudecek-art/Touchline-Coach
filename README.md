# Touchline Coach TC 1.6.0 – Role Foundation

Bod 1: Role

Připraveno:
- owner
- team_admin
- rt
- member
- pomocná funkce pro rozpoznání role podle prefixu kódu:
  - OWN-* = owner
  - ADM-* = admin týmu
  - RT-* = realizační tým
  - CL-* nebo HR-* = člen

Důležité:
- Tahle verze zatím NEMĚNÍ datovou strukturu.
- Neodděluje ještě týmy.
- Nepřidává cloud.
- Je to bezpečný základ pro TC 1.6.
